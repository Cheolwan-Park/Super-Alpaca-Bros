Basic Types

vec3

3d vector

x, y, z (float) : value of each element

vec2

2d vector

x, y (float) : value of each element

rect

float rectangle

x, y (float) : bottom left point's x, y element value
w, h (float) : width and height of the rect

irect

int rectangle

x, y (int) : bottom left point's x, y element value
w, h (int) : width and height of the rect

size

size (width and height)

w, h (int) : width and height

point

int 2d vector

x, y (int) : value of each element

Shader

shader information

name (string) : id of the shader
vert (string) : vertex shader filename
frag (string) : fragment shader filename

Animation

animation information

name (string) : id of the animation
texture (string) : texture filename of the animation
range (irect) : sprites rectangle used for animation
count (size) : sprites slicing in range rect
eachtime (float) : each sprite showing time
loop (bool) : is animation looping

ObjectStorage

storage for containing GameObjects

name (string) : id of the storage
order (int) : updating order (lower, earlier the storage updates)
size (int) : storage hash size

DrawableStorage

storage for containing Drawables and drawing them

name (string) : id of the storage
shader (string) : shader id for drawing items storage contains
order (int) : updating order (lower, earlier the storage drew)
size (int) : storage hash size

File Format

Initialize file

file for initialize game

title (string) : title of the game
windowsize (size) : size of the window
camera_range (size) : camera view size
allocator_size (int) : allocator memory size (unit: byte)
shaderstorage_size (int) : shaderstorage hash size
texturestorage_size (int) : texturestorage hash size
animationstorage_size (int) : animationstorage hash size
default_scene (string) : default scene file

Scene file

file for initialize scene

Shaders (array: Shader) : array of Shader
Textures (array: string) : array of texture filename
Animations (array: Animation) : array of Animation
ObjectStorages (array: ObjectStorage) : array of ObjectStorage
DrawableStorages (array: DrawableStorage) : array of DrawableStorage
GameObjects (array: GameObject) : array of GameObject

GameObjects

GameObject

Base class for all GameObject

type (string) : type of gameobject (classname)
storage (string) : name of the storage contains gameobject
id (string) : id of gameobject
available (bool) : is gameobject available (updated)
static (bool) : is gameobject static (not movable)
position (vec3) : location of gameobject
scale (vec2) : scale of gameobject
rotation (float) : rotation of gameobject

file (string) : init gameobject with file (file containing Object(GameObject))

ComponentObject

GameObject that can contain Components

type (string) : type of gameobject (classname)
storage (string) : name of the objectstorage contains gameobject
id (string) : id of gameobject
available (bool) : is gameobject available (updated)
static (bool) : is gameobject static (not movable)
position (vec3) : location of gameobject
scale (vec2) : scale of gameobject
rotation (float) : rotation of gameobject
components (array) : array of components gameobject contains

Components

Component

Base class for all Component (cannot instantiate)

type (string) : type of component (classname)
available (bool) : is component available (updated)

Drawable

Base class for drawing Component (cannot instantiate)

type (string) : type of component (Drawable)
available (bool) : is component available (updated)
id (string) : id of drawable component
storage (string) : name of the drawablestorage contains drawable (if none, not registering)

Sprite

Component for drawing sprite

type (string) : type of component (Sprite)
available (bool) : is component available (updated)
id (string) : id of drawable component
storage (string) : name of the drawablestorage contains drawable (if none, not registering)
uv (irect) : uv rect (unit: pixel)
texture (string) : texture id

AnimatedSprite

Component for sprite containing animation

type (string) : type of component (AnimatedSprite)
available (bool) : is component available (updated)
id (string) : id of drawable component
storage (string) : name of the drawablestorage contains drawable (if none, not registering)
uv (irect) : uv rect (unit: pixel)
texture (string) : texture id
default_anim (string) : default animation id

Alpaca

Alpaca Player Component

type (string) : type of component (Alpaca)
available (bool) : is component available (updated)
head (ComponentObject) : head object for alpaca
speed (float) : alpaca moving speed
keymap (int) : 0 -> 1P, 1 -> 2P
action_manager (ActionManager) : ActionManager for alpaca

Head

Alpaca's Head Component

type (string) : type of component (Head)
available (bool) : is component available (updated)
id (string) : id of drawable component
storage (string) : name of the drawablestorage contains drawable (if none, not registering)
uv (irect) : uv rect (unit: pixel)
texture (string) : texture id headpos (vec3) : default head position
neckpos (vec3) : default neck position
neck (ComponentObject) : neck object for alpaca

Extra types

ActionManager

Alpaca's Actions Manager class

actions (array: Action) : Actions alpaca contains
resettime (float) : time to reset combo

Action

Base class of Alpaca's Actions (Skill) class

type (string) : type of action (classname)
cooltime (float) : cooltime of action

Heading

Heading action

type (string) : type of action (classname)
cooltime (float) : cooltime of action
time (float) : taking time of heading action
length (float) : head flying length

Dash

Dash action

type (string) : type of action (classname)
cooltime (float) : cooltime of action
time (float) : taking time of dash action
length (float) : dashing length
